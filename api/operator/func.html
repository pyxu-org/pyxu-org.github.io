Returns:

NDArray

Notes

For \(\tau >0\), the proximity operator of a scaled functional \(f: \mathbb{R}^{M} \to \mathbb{R}\) is defined as:

\[\mathbf{\text{prox}}_{\tau f}(\mathbf{z}) := \arg\min_{\mathbf{x}\in\mathbb{R}^{M}} f(x)+\frac{1}{2\tau} \|\mathbf{x}-\mathbf{z}\|_{2}^{2}, \quad \forall \mathbf{z} \in \mathbb{R}^{M}.\]

class L21Norm(arg_shape, l2_axis=(0,))[source]#

Bases: _ShiftLossMixin, ProxFunc

Mixed \(\ell_{2}-\ell_{1}\) norm, \(\Vert\mathbf{x}\Vert_{2, 1} := \sum_{i=1}^{N} \sqrt{\sum_{j=1}^{M} x_{i, j}^{2}}\), for arrays of dimension \(\geq 2\).

Note

The input array need not be 2-dimensional: the \(\ell_{2}\) norm is applied along a predefined subset of dimensions, and the \(\ell_{1}\) norm on the remaining ones.

class HyperSlab(a, lb, ub)[source]#

Bases: _IndicatorFunction

Indicator function of a hyperslab.

\[\begin{split}\iota_{\mathbf{a}}^{l,u}(\mathbf{x}) := \begin{cases} 0 & l \le \langle \mathbf{a}, \mathbf{x} \rangle \le u \\ \infty & \text{otherwise}. \end{cases}\end{split}\]

class AffineSet(A, b)[source]#

Bases: _IndicatorFunction

Indicator function of an affine set.

\[\begin{split}\iota_{\mathbf{A}}^{\mathbf{b}}(\mathbf{x}) := \begin{cases} 0 & \mathbf{A} \mathbf{x} = \mathbf{b} \\ \infty & \text{otherwise}. \end{cases}\end{split}\]
\[\text{prox}_{\tau\, \iota_{\mathbf{A}}^{\mathbf{b}}}(\mathbf{x}) := \mathbf{x} - \mathbf{A}^{T} (\mathbf{A}\mathbf{A}^{T})^{-1} (\mathbf{Ax - b})\]

Notes

  • Assumptions on \(\mathbf{A}\) and \(\mathbf{b}\):

    • \(\mathbf{b} \in \text{span}(\mathbf{A})\).

    • \(\mathbf{A}\) has full row-rank, i.e. \(\mathbf{A}\) is square or fat.

  • AffineSet instances are not arraymodule-agnostic: they will only work with NDArrays belonging to the same array module as A and b.

class ConvexSetIntersection(*args)[source]#

Bases: _IndicatorFunction

Indicator function of an intersection of convex domains \(\mathcal{C}_{1} \cap \cdots \cap \mathcal{C}_{K}\).

\[\iota(\mathbf{x}) := \iota_{1}(\mathbf{x}) + \cdots + \iota_{K}(\mathbf{x}).\]

\(\text{prox}_{\tau\, \iota}(\mathbf{x})\) is computed using the PoCS algorithm. (Dykstra’s variant [PoCS_Dykstra].)

This function assumes \(\mathcal{C}_{1} \cap \cdots \cap \mathcal{C}_{K} \ne \emptyset\). prox() will loop indefinitely if this condition is violated.

Examples

import numpy as np
import pyxu.operator as pxo

op = pxo.ConvexSetIntersection(             # intersection of
    pxo.LInfinityBall(),                    #   L-\infty ball centered at (0,0)
    pxo.L2Ball().asloss(np.array([1, 0])),  #   L2 ball centered at (1,0)
)

x = np.array([
    [0, 0],
    [1, 0],
    [2, 0],  # not in LInfinity ball -> \infty
    [1, 1],
    [1, -1],
    [0, 1],  # not in L2 ball -> \infty
 ])
op.apply(x)  # [0 0 inf 0 0 inf]
